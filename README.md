# Book Master - Тестовое задание

Тестовое задание представляет собой REST API для ведения базы книг и их авторов. Приложение реализовано с использованием Spring Boot и PostgreSQL.

## Требования

- Docker
- Docker Compose
- Maven

## Запуск

1. Клонируйте репозиторий:
```shell
git clone git@github.com:WordyArc/book-master.git
```

2. Соберите jar файл приложения:
```shell
mvn clean package
```

3. Запустите Docker Compose:
```shell
docker-compose up --build
```

4. Приложение будет доступно по адресу http://localhost:8080.


## Swagger

Для просмотра документации API, перейдите по следующей ссылке после запуска приложения:

Swagger UI - http://localhost:8080/swagger-ui/index.html#/


## Стек

- **Spring Boot**: Основной фреймворк для разработки приложения.
- **Spring Data JPA**: Используется для работы с базой данных через JPA.
- **Spring Web**: Используется для разработки RESTful веб-сервисов.
- **Spring Validation**: Обеспечивает валидацию данных.
- **Flyway**: Для управления миграциями базы данных.
- **PostgreSQL**: Реляционная база данных.
- **Lombok**: Упрощает разработку за счет автоматического создания геттеров, сеттеров и других шаблонных методов.
- **Spring Boot DevTools**: Обеспечивает улучшения для разработки, такие как автоматическая перезагрузка.
- **SpringDoc OpenAPI**: Для генерации и отображения документации API.
- **Swagger Annotations**: Для аннотаций Swagger в коде.
- **ModelMapper**: Для маппинга между объектами.


## REST API

API включает следующие функции:

    Добавление новой книги: POST
    Изменение информации о книге, включая авторов: PUT
    Удаление книги: DELETE
    Получение списка книг с фильтрацией по наименованию, ISBN, автору: GET
    Добавление автора без привязки к книге: POST
    Изменение данных автора: PUT
    Удаление автора: DELETE
    Получение списка авторов: GET

## Описание сущностей

Book (Книга)

    id: Уникальный идентификатор книги
    title: Название книги
    isbn: Международный стандартный номер книги
    publicationDate: Дата публикации
    authors: Список авторов

Author (Автор)

    id: Уникальный идентификатор автора
    firstName: Имя автора
    lastName: Фамилия автора
    books: Список книг
